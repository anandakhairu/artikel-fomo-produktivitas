# 📊 Analisis Hubungan antara Fear of Missing Out (FOMO) dan Produktivitas Belajar di Kalangan Gen Z

Penelitian kuantitatif yang menganalisis hubungan antara 
FOMO dengan produktivitas belajar mahasiswa Gen Z.

## 📝 Abstrak
FOMO mempengaruhi produktivitas belajar Gen Z secara tidak 
langsung. FOMO mendorong intensitas penggunaan media sosial, 
dan intensitas yang berlebihan menekan produktivitas akademik.

## 🔍 Metodologi
- Pengumpulan data menggunakan Google Form
- Total responden: 189 mahasiswa Gen Z
- Pendekatan kuantitatif

## 🛠️ Tools
- Google Forms
- Analisis Statistik
- Penelitian Kuantitatif
